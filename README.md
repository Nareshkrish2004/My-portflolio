# Naresh K — UI/UX Designer Portfolio

A modern, editorial-style React portfolio inspired by Swiss modernist design.

## 🎨 Design System
- **Colors:** Black (#0A0A0A) · Off-white (#F5F4EF) · Yellow accent (#F0E040)
- **Fonts:** Bebas Neue (display) · Space Mono (mono) · DM Sans (body)
- **Style:** Swiss editorial / bold modernist

## 🚀 Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view in the browser.

## 📦 Build for Production

```bash
npm run build
```

## 📂 Project Structure
```
src/
  App.js       — All components (Nav, Hero, About, Skills, Experience, Projects, Services, Contact, Footer)
  App.css      — Complete stylesheet with CSS variables, animations, responsive design
  index.js     — React entry point
public/
  index.html   — HTML shell with Google Fonts
```

## ✏️ Customization
- Update contact details in the `Contact` component
- Add real project images by replacing the `proj__thumb` placeholder divs
- Adjust skill percentages in the `skills` array
- LinkedIn URL in `Contact` and `Footer` components
